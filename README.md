Product Management System

A full-stack web application for managing products with complete CRUD (Create, Read, Update, Delete) functionality. The system is designed with a clean architecture, RESTful APIs, and a responsive frontend interface.

Overview

The Product Management System provides a structured and intuitive way to manage product data. It enables users to create, update, delete, and view products through a seamless frontend interface connected to a robust backend.

This project demonstrates:
End-to-end full-stack development
RESTful API design and implementation
Database integration and data persistence
Clean separation between frontend and backend 

Tech Stack:
Frontend->
React (Vite)
HTML, CSS, JavaScript
Tailwind CSS 

Backend->
Spring Boot (Java)
REST APIs
MVC architecture

Database->
MySQL or PostgreSQL


Key Features:
Create, update, and delete products
View all products in a structured format
Retrieve product details by ID
Clean and responsive user interface
Efficient API communication between frontend and backend

API Overview:
Product Endpoints:
GET /products- Retrieve all products
GET /products/{id}- Retrieve a product by its ID
POST /products- Create a new product
PUT /products/{id}- Update an existing product
DELETE /products/{id}- Delete a product
